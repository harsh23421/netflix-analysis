# 🎬 Netflix Data Analysis

A complete **Exploratory Data Analysis (EDA)** project on the Netflix Movies & TV Shows dataset using Python. This project uncovers trends, patterns, and insights about Netflix's content library through data cleaning, visualization, and statistical analysis.

---

## 📌 Project Overview

The objective of this project is to analyze Netflix's catalog and answer business-related questions such as:

- How has Netflix's content grown over the years?
- Which countries produce the most Netflix content?
- What is the ratio of Movies to TV Shows?
- Which genres dominate the platform?
- How has content production changed over time?
- What are the most common ratings assigned to content?

The project demonstrates the complete Data Analysis workflow:
- Data Collection
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insights & Conclusions

## 📌 Key Insights

Some important findings from the analysis include:

- Netflix has significantly increased its content library after 2015.
- Movies make up a much larger portion of the catalog than TV Shows.
- The United States contributes the highest number of titles.
- Drama and International Movies are among the most popular genres.
- TV Shows generally span fewer seasons.
- Most content is targeted toward mature audiences (TV-MA).
